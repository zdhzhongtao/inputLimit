输入框数字控制插件

版本号2.0.4

配置属性 可配置属性 用途 默认值
type number、int、letters、number_letters 只能输入数字和小数点只能输入整数只能输入大小写英文字符只能输入数字和大小写英文字符 numnber
negative 布尔值 是否可以输入负号 true
first 整数或者normal 小数点前面位数 8
last 整数或者normal 小数点后面位数 2
includeZero 布尔值 是否包括0 true
underline 布尔值 设置在使用控制数字字母的时候是否可以输入下划线 true
max 整数或者normal 设置最大值 normal
callback function(thisEle,val) 回调函数(当前作用的节点jquery对象,改变后的数字) 空

注意事项

  • 在使用控制输入只能输入数字和英文字母时，下划线的设置不生效
  • 设置英文字母和数字的时候还可以使用下划线，这个一般用于账号的输入内容,如果没有下划线，则使用underline: false
